# Country Explorer (React)

This folder contains the "Country Explorer" assignment, a frontend application built using modern React to display and explore country data.

## Technologies Used

*   **React (v19)**: A JavaScript library for building user interfaces. It allows developers to create large web applications that can change data, without reloading the page. Its main goal is to be fast, scalable, and simple.
*   **Vite**: A modern frontend build tool that significantly improves the frontend development experience. It provides a faster and leaner development environment compared to older tools like Create React App or Webpack.
*   **Tailwind CSS**: A utility-first CSS framework packed with classes like `flex`, `pt-4`, `text-center` and `rotate-90` that can be composed to build any design, directly in your markup.
*   **React Hook Form**: A library that helps manage complex forms in React. It improves performance by minimizing the number of re-renders caused by form state changes and makes form validation much easier.

## Core Concepts Covered

*   **Components**: Independent, reusable pieces of code that serve as the building blocks of any React application. They return HTML via a syntax extension called JSX.
*   **Hooks**: Functions that let you "hook into" React state and lifecycle features from functional components. Common examples include `useState` for managing local state and `useEffect` for handling side effects like data fetching.
*   **Form Handling**: Managing user inputs, capturing data, and executing validation rules efficiently without sacrificing application performance.

## How to Run

1.  Open your terminal and navigate to this folder.
2.  Install the required dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the provided `localhost` URL in your browser to view the application.
